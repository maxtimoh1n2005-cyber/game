# ColorMatch Duel - Telegram Mini App

## 🚀 Быстрый старт

### 1. Загрузи файлы на хостинг
Загрузи все файлы (index.html, style.css, script.js, manifest.json) на любой HTTPS-хостинг:
- GitHub Pages
- Netlify
- Vercel
- Любой другой

### 2. Создай бота в Telegram
1. Найди @BotFather
2. Отправь `/newbot` и создай бота
3. Отправь `/newapp` или используй `/setdomain` для настройки Mini App
4. Укажи URL твоего приложения

### 3. Готово!
Твое приложение доступно через бота!

## 📱 Как играть
- Два игрока соревнуются по очереди
- За 3 секунды нужно запомнить цвет
- С помощью ползунков воссоздать точный цвет
- Побеждает тот, у кого выше точность после 5 раундов
